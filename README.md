# VoteTally Pro - 学生会部长竞选计票系统

一个极简、高效的现代化计票工具，专为学生会竞选设计。

## 功能特点

- **批量导入**: 支持 TXT 文件导入候选人名单，格式灵活。
- **实时计票**: 现代化的大屏展示，实时更新票数。
- **快速投票**: 
  - 支持拼音首字母搜索 (如 `zs` -> 张三)。
  - 全键盘操作支持 (方向键选择，回车确认)。
- **结果公示**: 自动生成精美的胜选结果报表，支持平票处理。

## 技术栈

- React + TypeScript
- Vite
- Tailwind CSS
- Lucide Icons

## 快速开始

1. 安装依赖:
   ```bash
   npm install
   ```

2. 启动开发服务器:
   ```bash
   npm run dev
   ```

3. 构建生产版本:
   ```bash
   npm run build
   ```

## 使用说明

1. **准备名单**: 准备一个 TXT 文件，每行一个候选人信息，或者直接复制粘贴。
   - 格式示例: `学习部 - 张三 (高二1班)`
2. **导入数据**: 在首页点击“导入 TXT”或粘贴名单。
3. **开始投票**: 进入投票页面，使用鼠标点击或键盘快捷键进行计票。
4. **完成统计**: 点击“完成统计”查看最终结果。
